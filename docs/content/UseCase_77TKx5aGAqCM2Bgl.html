Scenario 1: Crear
Steps
1. ADMINISTRADOR selecciona el opción de modalidades en el menú principal
2. SYSTEM muestra modulo de modalidades
3. ADMINISTRADOR da clic en el botón crear
4. SYSTEM muestra formulario para creación de modalidad
5. ADMINISTRADOR diligencia el formulario y da clic en botón guardar
6. SYSTEM valida y guarda la información

Extension
Steps
5.a. botón "Cancelar"
    1. ADMINISTRADOR da clic en botón cancelar
    2. SYSTEM muestra modulo de modalidades

Scenario 2: Modificacion
Steps
1. ADMINISTRADOR selecciona el opción de modalidades en el menú principal
2. SYSTEM muestra modulo de modalidades
3. ADMINISTRADOR selecciona el modulo a modificar y da clic en botón modificar
4. SYSTEM muestra formulario para modificación de modalidades
5. ADMINISTRADOR modifica datos de modalidad da clic en botón guardar
6. SYSTEM valida y guarda la información

Extension
Steps
4.a. botón "Cancelar"
    1. ADMINISTRADOR da clic en botón cancelar
    2. SYSTEM muestra modulo de modalidades

Scenario 3: Eliminar
Steps
1. ADMINISTRADOR selecciona el opción de modalidades en el menú principal
2. SYSTEM muestra modulo de modalidades
3. ADMINISTRADOR selecciona modalidad y da clic el botón de eliminar
4. SYSTEM muestra pantalla ventana de confirmación
5. ADMINISTRADOR selecciona botón si
6. SYSTEM elimina modalidad

Extension
Steps
4.a. botón "No"
    1. ADMINISTRADOR selecciona botón no
    2. SYSTEM muestra modulo de modalidades

Scenario 4: Consultar
Steps
1. ADMINISTRADOR selecciona el opción modalidades en el menú principal
2. SYSTEM muestra modulo de modalidades
3. ADMINISTRADOR selecciona el botón de consultar
4. SYSTEM muestra ventana de consulta
5. ADMINISTRADOR ingresa búsqueda y da clic en botón buscar
6. SYSTEM muestra resultado de búsqueda
