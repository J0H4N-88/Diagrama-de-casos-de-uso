Scenario 1: Crear
Steps
1. PROVEEDOR oADMINISTRADOR oANONIMO da clic en datos personales
2. SYSTEM muestra modulo datos personales
3. PROVEEDOR oADMINISTRADOR oANONIMO da clic en crear
4. SYSTEM muestra formulario de diligenciamiento
5. PROVEEDOR oADMINISTRADOR o ANONIMO diligencia formato y da clic en guardar
6. SYSTEM valida y guarda informacion

Extension
Steps
4.a. botón "Cancelar"
    1. PROVEEDOR oADMINISTRADOR da clic en cancelar
    2. SYSTEM muestra modulo datos personales

Scenario 2: Modificar
Steps
1. PROVEEDOR o ADMINISTRADOR o CLIENTE da clic en perfil de usuario
2. SYSTEM muestra modulo perfil de usuario
3. PROVEEDOR oADMINISTRADOR o CLIENTE modifica la informacion que requiera
4. SYSTEM valida y guarda informacion

Extension
Steps
3.a. botón "Cancelar"
    1. PROVEEDOR oADMINISTRADOR da clic en cancelar
    2. SYSTEM muestra modulo datos personales

Scenario 3: Eliminar
Steps
1. ADMINISTRADOR ingresa modulo base de datos de plataforma
2. SYSTEM muestra modulo datos
3. ADMINISTRADOR elimina la informacion que requiera
4. SYSTEM valida y guarda informacion

Extension
Steps
3.a. botón "Cancelar"
    1. PROVEEDOR oADMINISTRADOR da clic en cancelar
    2. SYSTEM muestra modulo datos personales
