Scenario 1: Crear
Steps
1. ADMINISTRADOR selecciona opción de vinculaciones en el menú principal
2. SYSTEM muestra modulo de vinculaciones
3. ADMINISTRADOR da clic en el botón crear
4. SYSTEM muestra formulario para creación de la vinculación
5. ADMINISTRADOR diligencia el formulario y da clic en guardar
6. SYSTEM valida y guarda la información

Extension
Steps
4.a. botón "Cancelar"
    1. ADMINISTRADOR selecciona botón cancelar
    2. SYSTEM muestra modulo de vinculaciones

Scenario 2: Eliminar
Steps
1. ADMINISTRADOR selecciona opción de vinculaciones en el menú principal
2. SYSTEM muestra modulo de vinculaciones
3. ADMINISTRADOR selecciona vinculación a eliminar y da clic en eliminar
4. SYSTEM muestra ventana de confirmación
5. ADMINISTRADOR selecciona la opción si
6. SYSTEM elimina el tipo de vinculación

Extension
Steps
4.a. botón "No"
    1. ADMINISTRADOR selecciona la opción no
    2. SYSTEM muestra modulo de vinculaciones

Scenario 3: Modificar
Steps
1. ADMINISTRADOR selecciona opción de vinculaciones en el menú principal
2. SYSTEM muestra modulo de vinculaciones
3. ADMINISTRADOR selecciona vinculación a modificar y da clic en modificar
4. SYSTEM muestra formulario para modificación de vinculación
5. ADMINISTRADOR realiza modificaciones y da clic en guardar
6. SYSTEM sistema valida y guarda la información

Extension
Steps
4.a. botón "Cancelar"
    1. ADMINISTRADOR da clic en cancelar
    2. SYSTEM muestra modulo de vinculaciones

Scenario 4: Consultar
Steps
1. ADMINISTRADOR selecciona opción de vinculaciones en el menú principal
2. SYSTEM muestra modulo de vinculaciones
3. ADMINISTRADOR selecciona el botón de consultar
4. SYSTEM muestra ventana de consulta
5. ADMINISTRADOR ingresa parámetro de búsqueda y da clic en buscar
6. SYSTEM muestra resultado de búsqueda
